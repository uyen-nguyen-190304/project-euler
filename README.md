# Project Euler - Solutions Archive

This repository contains my solutions to the Project Euler problems, along with notes explaining my reasoning and approach for each problem. 

Since Project Euler specifically asks that solutions of problems, except for the first 100, not to be shared publicly, I will continue to update the repository with my solutions and notes up to problem 100. Beyond that point, all solutions and notes will be kept private, while this repo will still remain public for the first 100 problems.


It is split into two parts:

- [problems](problems) for the Python solutions
- [notes](notes) for the solution writeups and reasoning
